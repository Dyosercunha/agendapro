 Arquivo historico de SQL
=========================

Esta pasta guarda scripts antigos de atualizacao do Supabase que ja foram usados durante a evolucao do AgendaPro.

Eles ficam aqui apenas como historico e referencia. Para novas alteracoes de banco, prefira criar um script novo em
`supabase/migrations/` ou um arquivo SQL novo com nome claro antes de executar no painel do Supabase.

Nao rode estes arquivos antigos novamente sem revisar o conteudo e confirmar que a alteracao ainda faz sentido para o
estado atual do banco.
