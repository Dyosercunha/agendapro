# AgendaPro

App de agendamento para barbearias e saloes, com painel de gerenciamento, agenda real, configuracoes por barbearia e integracao com Supabase.

## Rodar o projeto

```bash
npm install
npm run dev
```

## Arquivos principais

- `App.tsx`: aplicacao React completa.
- `styles.css`: visual premium do app.
- `supabase_schema.sql`: estrutura inicial do banco.
- `supabase_update_*.sql`: atualizacoes incrementais do Supabase.
